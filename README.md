# foundry

a command line client to share folders and chat

requires an api environment variable, see accounts.json for latest

* DEEPSEEK_API_KEY
* OPENAI_API_KEY 
* XAI_API_KEY 
* GEMINI_API_KEY

# interface

[model modelname promptTokens replyTokens totalTokens contextSize]

# commands

/model [id]

/share [folder]

/push

/reset
